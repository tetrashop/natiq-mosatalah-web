راهنمای پاکسازی کش مرورگر

  1. Chrome: Settings → Privacy → Clear browsing data → Cached images and files
  2. Firefox: Settings → Privacy & Security → Clear Data → Cached Web Content
  3. Safari: Settings → Advanced → Website Data → Remove All
  4. روش سریع: Ctrl+Shift+R (دسکتاپ) یا کشیدن صفحه به پایین (موبایل)
